    NTT株式会社は3月27日、5Gや6Gをはじめとするさまざまな無線環境がある屋内や屋外の窓ガラスに設置することで、電波を制御できる「透過型メタサーフェスデバイス」を開発したと発表した。建物の窓ガラスなどに貼付すれば、透過する電波を制御して、遮蔽物などによって、電波が届きにくい場所の電波環境を改善できるという。...
