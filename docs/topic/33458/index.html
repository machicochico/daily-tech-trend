    高架橋の鋼橋に亀裂16カ所、緊急設置 名古屋市の国道23号
    NIKKEI TECH

    名古屋市港区の国道23号竜宮高架橋の鋼桁に最大で長さ約30cmの亀裂が16カ所見つかった。国土交通省名古屋国道事務所は橋を通行止めにし、当て板を添えるなど応急対策を進めている。2021年10月に実施した前回の定期点検では主桁に亀裂は見られず、健全度が4段階で良好な方から2番目のIIだった。同事務所は、多くの大型車の通行に伴う疲労亀裂と見ている。...
