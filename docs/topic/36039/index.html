    分身ロボットを遠隔で操作案内業務を行うパイロットを募集します
    東京都 報道発表

    東京都では、重度障害のある方等の能力を十分に発揮することができるよう、遠隔操作が可能な分身ロボットを活用した新たな働き方の普及啓発を図る事業を行っています。 本年は、公益財団法人東京しごと財団に、自宅等から遠隔操作できる分身ロボット「OriHime」を設置します。「OriHime」を操作し、東京しごと財団にて案内業務等を行う重度障害のある方等を募集します。 ...
