    インドネシアでエビ養殖事業に参入、オリコングローバル
    NIKKEI TECH

    オリエンタルコンサルタンツグローバル（OCG）が、インドネシアで水産物（バナメイエビ）の養殖事業に乗り出す。事業を手掛ける特別目的会社を共同出資で設立。同国中部のスラウェシ島にエビの養殖施設を整備し、年間1300トンの供給体制を構築する。これまで、同社の海外事業は主に技術支援や調査・計画などの受託型が中心だったが、資本参画する投資型の事業モデルを展開する。...
