1. 注意喚起: 2026年7月マイクロソフトセキュリティ更新プログラムに関する注意喚起

何が起きているか
Microsoftが2026年7月にリリース予定のセキュリティ更新プログラムに関し、既知の脆弱性が修正されていない環境への影響を警告。
事業への影響
社内システムや顧客向けサービスでMicrosoft製品を利用している企業は、更新プログラムの適用遅延により攻撃リスクが増大し、業務停止やデータ漏えいの可能性が高まる。
推奨行動
即時にパッチ適用計画を策定し、テスト環境での検証後に本番環境へ展開。更新遅延が発生した場合は代替対策（ファイアウォール設定強化等）を講じ、経営層へ進捗報告を行う。

2. Linuxカーネルのx86向けKVMに解放済みメモリの使用の脆弱性

何が起きているか
KVM仮想化環境で解放済みメモリを不正に利用できる脆弱性が報告され、ゲストVMからホストへのDoSや任意コード実行が可能となる。
事業への影響
クラウドサービスプロバイダーや社内仮想化環境を持つ企業は、サービス停止や機密情報漏えいのリスクが急増。
推奨行動
各ディストリビューションのパッチ適用状況を確認し、未適用の場合は直ちに更新。仮想化環境の監査を実施し、脆弱性利用検知機能を導入。経営層へリスク評価と対策計画を報告。

3. CISA、悪用された既知の脆弱性 4 件をカタログに追加

何が起きているか
米国CISAが4件の既知脆弱性（CVE-2026-15409等）をKEVカタログに追加し、実際に悪用が確認されたことを公表。
事業への影響
組織はこれらの脆弱性を対象とした攻撃を受ける可能性が高く、特にSharePointやICS製品を利用している企業は業務継続に重大な影響を受ける。
推奨行動
KEVリストにある脆弱性を対象に、脆弱性管理プロセスを強化。パッチ適用優先度を上げ、必要に応じてネットワーク分離やアクセス制御を再設定。経営層へリスクマトリクスを提示し、緊急対策資源の確保を提案。
