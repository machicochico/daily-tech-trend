3. CISAが発表した複数のICS/IIoT脆弱性（Siemens、Rockwell Automation等）

何が起きているか
米国CISAがSiemens、Rockwell Automation、Palo Alto Networks製品等に対する複数の脆弱性を公開。攻撃者は認証バイパス、権限昇格、DoS等を実行可能。
事業への影響
工場・インフラ運用に依存する事業では、既存機器の脆弱性が即座にリスクとなり、稼働停止やデータ漏洩、法規制違反の可能性が高まる。顧客からの信頼低下や訴訟リスクも増大。
推奨行動
セキュリティ担当者は直ちに対象機器のパッチ適用状況を確認し、未適用の場合は緊急パッチ適用を実施。運用マニュアルを更新し、脆弱性対策の定期的な監査を導入。顧客へのリスク情報共有と対応計画を策定し、ビジネス継続計画に組み込む。
