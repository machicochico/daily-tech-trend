1. EU、Metaのインスタ・フェイスブックの無限スクロール等をデジタルサービス法違反と見なす予備的見解

何が起きているか
EU委員会がMeta社のInstagram・Facebookにおける無限スクロール、プッシュ通知、自動再生等のデザインが、ユーザーの過度な利用を誘発し、デジタルサービス法（DSA）に違反すると判断し、規制強化の可能性を示唆した。
事業への影響
日本企業がMetaプラットフォームを広告・マーケティングに利用している場合、規制対象となる可能性が高まり、広告配信制限や追加のコンプライアンスコストが発生。さらに、ユーザーエンゲージメントの低下やブランドイメージへの影響も懸念される。
推奨行動
① 既存広告戦略を見直し、過度なエンゲージメントを誘発しないコンテンツ設計を検討。② 規制リスクを評価し、必要に応じて広告配信量・頻度を調整。③ 監査・コンプライアンス体制を強化し、DSAに準拠したレポーティングを実施。

2. JPCERT/CC、LINE iOS版アプリのDoS脆弱性を報告

何が起きているか
LINEヤフー株式会社が提供するiOS版LINEアプリ内ブラウザに、任意URLスキーム処理時の保護が不十分でサービス停止を引き起こす脆弱性が存在し、JPCERT/CCがアドバイザリを発表した。
事業への影響
LINEを主要コミュニケーションツールとして利用する企業は、脆弱性が悪用されることで業務連絡や顧客対応が停止し、業務継続に支障をきたすリスクがある。加えて、顧客情報漏えいの可能性も懸念される。
推奨行動
① 速やかにLINEアプリの最新版へアップデートし、脆弱性パッチを適用。② 社内でLINE利用に関するセキュリティポリシーを策定し、従業員への教育を実施。③ 外部監査や脆弱性スキャンを定期的に実施し、リスクを継続的に管理。

3. 住信SBIネット銀行、NTTドコモのdアカウントと連携開始

何が起きているか
住信SBIネット銀行が、NTTドコモの共通ID「dアカウント」と連携し、2023年8月20日から銀行サービスへのログインや本人確認を一元化した。
事業への影響
日本の金融機関は、共通ID連携により顧客利便性が向上し、顧客獲得・維持に有利になる一方、ID管理・認証プロセスの統合に伴うセキュリティリスクや運用負荷が増大する。
推奨行動
① 自社の認証・ID管理システムを見直し、dアカウント連携に対応できるよう設計。② セキュリティリスク評価を実施し、二段階認証や監査ログの整備を強化。③ 顧客への導入メリットを訴求し、利用促進キャンペーンを実施。
