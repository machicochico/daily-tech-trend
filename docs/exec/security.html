1. ルーターの衛生状態を改善してロシア国家支援の標的から守る (CISA Cybersecurity Advisories)

何が起きているか
ロシア政府支援のサイバーアクターが、設定ミスや脆弱性のあるルーターを標的に攻撃を行う事例が報告され、CISAが対策を推奨。
事業への影響
通信機器メーカーやIoTデバイスメーカーは、製品の初期設定やファームウェア更新プロセスを見直し、顧客へのセキュリティ教育を強化しないと、信頼失墜や法的責任、売上減少につながる。
推奨行動
①製品設計段階でセキュリティベースラインを設定し、初期設定を安全な状態に固定。②ファームウェアアップデートを自動化し、定期的な脆弱性スキャンを実施。③顧客向けマニュアルやオンラインサポートでセキュリティ対策を啓発し、サポート体制を整備。

2. セブン、テスラ「スーパーチャージャー」導入

何が起きているか
セブン-イレブン・ジャパンがテスラの急速充電設備を店舗に導入し、EVユーザー向けサービスを拡充。
事業への影響
飲食業・小売業の新規収益源となり、EV普及に伴う顧客層拡大が期待できる。競合他社も同様の導入を検討することで市場シェア争いが激化。
推奨行動
①自社店舗の電力インフラを評価し、充電設備導入の可否を判断。②充電料金設定とプロモーション戦略を策定し、顧客誘致を図る。③充電設備の運用・保守体制を確立し、顧客満足度を高める。

3. 店舗向けステーブルコイン決済「Stablecoin Pay」本格始動

何が起きているか
ネットスターズが店舗向けに複数ステーブルコイン決済を可能にするサービスを開始。
事業への影響
キャッシュレス決済の選択肢が増え、海外顧客やデジタル資産保有者の来店を促進。決済手数料が低減される一方、規制リスクやマネーロンダリング対策が課題。
推奨行動
①自社決済システムにステーブルコイン対応を検討し、導入コストとリスクを評価。②規制遵守体制（AML/KYC）を整備し、取引監視を強化。③顧客への教育・プロモーションを実施し、デジタル資産利用者の獲得を狙う。
